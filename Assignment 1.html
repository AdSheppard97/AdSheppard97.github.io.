The github workflow

To begin with a git commit is like saving a change to a document or project but it records who made the change and what has changed inside.

A git branch is like copying and pasting a project for you to work on without actually making any changes to the repository and is a safe way to change code.

Finally a git pull request is like a review before you commit to the main repository

The Unix Shell

The Unix Shell or bash is a very powerful tool used for file management outside of your standard OS file management system The tool primarily uses commands similar to a command promptcmd including but not limited to ls which lists all of your files in the targeted directory pwd> which shows you the current directory you are viewing/editing additionally you can use cd or cd dir which changes your targeted directory.
